    JavaScript is a synchronous single-threaded language. This means JS can only execute one line at a time and in a specific order

    Execution Context

    Everything in JavaScript happens inside an “Execution Context”. So whenever a program runs in javascript, the javascript engine creates a Global execution context for that and pushes that into the call stack. And also whenever we invoke a function in javascript, an “Execution Context” is created for that in the global execution context. And pushes that into the call stack
